# Architecture-Microservices-TP2
Ce TP nous a permis de maîtriser les services web SOAP avec JAX-WS. Nous avons créé un service bancaire avec publication de WSDL, utilisé les annotations JAXB pour contrôler le XML généré, testé avec SoapUI, et développé un client pour consommer le service. Cette activité couvre tout le cycle de vie d'un service web SOAP.
